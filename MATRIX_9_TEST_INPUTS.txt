Policy Integrity 3x3 Matrix Test Plan (9 Demo Outcomes)

Goal
Show all 9 combinations:
- Policy outcomes: VALIDATED, CORRECTED, BLOCKED
- Governance modes: Validation Only, Human-in-the-Loop, Full Automation

Where to run
- Adoption Studio: set governance profile
- Policy Workbench: create policy + run
- Correction Center: approve/resolve where required

One-time setup
1) Open Adoption Studio.
2) Keep Auto-Apply Minimum Confidence at default (90%) unless noted.
3) Open Policy Workbench.
4) For each case below: fill form values, click Create Policy, then click Run.
5) Record final lifecycle status from Policies table.
